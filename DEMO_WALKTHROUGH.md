# YalOffice Candidate Demo Walkthrough

Follow this script to record a complete "Happy Path" demo of the candidate journey.

## Prerequisites
1. Ensure the application is running at `http://localhost:3001`.
2. Ensure you have a simple resume file (PDF or Text) ready for upload, or use the `dummy_resume.txt` provided in the root.

## Scene 1: Usage & Application
1. **Navigate** to `http://localhost:3001`.
   - *Action*: If logged in, click "Logout" to start fresh.
2. **Sign Up**:
   - Click **"Sign Up"**.
   - **Name**: "Alex Candidate"
   - **Email**: `alex.demo.01@example.com` (Use a unique email)
   - **Role**: Candidate
   - **Mobile**: (Any 10 digit number)
   - Click **Sign Up**.
3. **Dashboard**:
   - Verify you land on the **Candidate Dashboard**.
   - Note the sections: "My Applications", "Recommended Jobs".
4. **Find a Job**:
   - Click **"Find Jobs"** in the sidebar.
   - Select a job (e.g., "Senior Software Engineer").
   - Click **"Apply Now"**.
5. **Application Form**:
   - **Resume**: Upload your resume file.
   - **Notice Period**: "Immediate".
   - **Source**: "LinkedIn".
   - Click **"Submit Application"**.
   - *Result*: You should be redirected/shown a success message.

## Scene 2: The Interview
1. **Navigate** to **"My Applications"**.
   - Find the job you just applied to.
   - The status should be 'Applied' or 'Screening'.
2. **Start Interview**:
   - If an **"Interview"** or **"Screening"** button is visible, click it.
   - *Note*: If not visible, the system might be processing. Refresh the page.
   - *Tech Check*: Allow microphone access if prompted.
3. **Live Interview**:
   - Click **"Start Call"** / **"Join Room"**.
   - Speak a few sentences to the AI Agent (e.g., "Hi, I'm Alex. I have 5 years of experience in React and Node.js.").
   - *Visuals*: Point out the real-time visualization or avatar.
   - When finished, click **"End Call"** or wait for the agent to finish.

## Scene 3: The Results
1. **Post-Interview Processing**:
   - Wait for the "Analyzing" screen to complete.
2. **View Report**:
   - You should be redirected to the **Interview Report**.
   - **Highlight**:
     - The **AI Score** (e.g., 85/100).
     - The **Summary**.
     - The **Pros/Cons** analysis.
3. **Conclusion**:
   - Return to Dashboard.
   - Show the application status updated to "Interviewed" or "Shortlisted".
